Choose a pre-evolved creature








If it falls over, restart:



(You may need to restart a few times)

You can also pick it up from the top and (try) to stand it up. Good luck.

You can run the genetic algorithm to evolve a new locomotion strategy with a population of creatures based on the chosen creature type.



When the genetic algorithm is done, it will show the most fit individual in the population. Or you can stop at any time and see it.



There are few bugs that need to be squashed. For now, refresh the browser before running another GA session. More controls and features are planned, but don't hold your breath. This code will be made available on github.
